************
Installation
************

Requirements
============

- Python 3.*
- pip 20.0.2
- torch 2.6.0

Installing by using PyPI
========================

Install
-------
.. code-block:: bash

	git clone https://github.com/intsystems/DFDistill /tmp/DFDistill
	python -m pip install /tmp/DFDistill/dfdistill/

Uninstall
---------
.. code-block:: bash

	python3 -m pip uninstall dfdistill
